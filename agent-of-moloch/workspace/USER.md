# User

The expected user is a DAO operator, guild member, or builder who wants an agent to help monitor Moloch/Baal proposals and maintain a durable voter platform.

They probably want to:

- add the DAOs the agent participates in
- record each DAO's charter, thesis, and governance constraints
- define the agent's conviction and voting platform
- inspect proposal records and voting recommendations
- ask the agent to prepare Moloch read, proposal, sponsor, vote, or process tasks
- keep the final broadcast decision human-controlled

Ask short clarifying questions only when a missing answer would affect a write, a vote recommendation, or an onchain action.

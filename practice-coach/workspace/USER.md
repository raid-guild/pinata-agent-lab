# User

The target user is a learner practicing music, writing, fitness, language, drawing, or another repeatable discipline.

The user values fast chat logging, clear feedback, visible progress, and knowing exactly what to practice next. They should not need to edit database records or fill out browser forms.

Jobs to be done:

- See current training status at a glance.
- Know which practice blocks are due.
- Review recent sessions and reflections.
- Ask the agent to log practice after a focused block.
- Ask the agent to store a next-session plan before stopping.

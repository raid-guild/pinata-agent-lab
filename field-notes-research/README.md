# Field Notes Research Agent

A Pinata-ready research notebook template for capturing observations, sources, tags, quotes, themes, follow-up questions, and a report-ready summary draft.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000/app`.

## Validate

```bash
npm run typecheck
npm run build
```

The SQLite database is created at `data/field-notes-research.sqlite` on first use.

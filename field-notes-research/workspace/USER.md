# User

The target user is a researcher, community operator, product lead, strategist, or founder who needs to preserve raw observations and turn them into usable themes.

The user wants a low-friction research partner in chat, not a heavy qualitative-analysis tool. They may paste call notes, interview fragments, Discord excerpts, support patterns, or desk research and ask the agent to structure the evidence.

The user values:

- Source traceability.
- Report-ready evidence.
- Clear tags and themes.
- Follow-up questions that improve the next conversation.
- A dashboard that is easy to scan without risking accidental edits.

When the user gives ambiguous notes, ask only for the missing fields needed to avoid bad data. When a reasonable default is obvious, state it and proceed.

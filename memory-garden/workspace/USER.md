# User

The target user is an independent thinker, writer, researcher, founder, or operator who wants ideas and reflections to remain findable over time.

The user values quick capture, visible topic clusters, and practical resurfacing over elaborate taxonomy.

Job to be done:

- Capture a thought quickly in chat without deciding on a perfect filing system.
- Revisit older ideas before they disappear from attention.
- See which clusters are active and which memories are connected.
- Turn repeated reflections into a usable personal knowledge map.

Useful setup preferences to learn:

- Preferred topic clusters.
- Whether notes should stay raw, lightly edited, or synthesized.
- What counts as a reviewed memory.
- Whether links should be sparse and high-confidence or more exploratory.

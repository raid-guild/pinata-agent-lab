# Community Quest Board

A Pinata-ready coordination template for cohorts, guilds, and project groups that need a simple board for quests, claims, updates, outcomes, and weekly recaps.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000/app`.

## Validate

```bash
npm run typecheck
npm run build
```

The SQLite database is created at `data/community-quest-board.sqlite` on first use.

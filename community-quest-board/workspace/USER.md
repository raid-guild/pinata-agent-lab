# User

Primary users are community stewards, cohort leads, guild coordinators, maintainers, and contributors who need a shared view of available, claimed, reviewed, and completed work.

They should not need to manage the database or edit records in the browser. They should be able to say things like:

- "Claim the onboarding blockers quest for Mira."
- "Add an update that the demo recap is ready for review."
- "Move the sponsor thank-you quest to done and record the approved quote outcome."
- "What still needs an owner this week?"

The dashboard answers status questions at a glance. Chat handles capture, cleanup, and coordination changes.
